# StandMuse — акустические стойки и аудио-решения

## Проект
Премиальный интернет-магазин по изготовлению акустических стоек, HI-FI стоек, панелей, диффузоров и аксессуаров на заказ.

## Стек
- Next.js App Router
- TailwindCSS
- Radix UI
- MongoDB + Prisma
- Redux Toolkit
- Темы: светлая/тёмная (auto-detect)
